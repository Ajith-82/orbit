Chatbot Widget Demo

This page demonstrates how to integrate the chatbot widget into your website.

You should see a chat icon in the bottom right corner. Click it to open the chat!

How to integrate

Add the following code to your website:

<!-- Include React and ReactDOM first -->
<script src="https://unpkg.com/react@18/umd/react.production.min.js"></script>
<script src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"></script>
<!-- Include the widget -->
<script src="https://cdn.example.com/chatbot-widget.umd.js"></script>
<link rel="stylesheet" href="https://cdn.example.com/style.css">

<script>
  // Initialize after everything is loaded
  setTimeout(function() {
    if (window.initChatbotWidget) {
      initChatbotWidget({
        apiUrl: 'https://your-api-server.com'
      });
    }
  }, 500);
</script>
